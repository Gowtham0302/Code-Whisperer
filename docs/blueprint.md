# **App Name**: Code Whisperer

## Core Features:

- Code Input: Clean and modern input area for users to paste or type code.
- Code Explanation: Utilize an AI model to analyze and generate explanations for the provided code.
- Explanation Display: Display the code explanation in a conversational format, as if a tutor is explaining it.
- Explanation Customization: Allow users to specify the desired level of detail in the explanation using a simple prompt. The AI tool should adjust the explanation accordingly.
- Syntax Highlighting: Provide syntax highlighting for the code input area to improve readability.

## Style Guidelines:

- Primary color: White or light grey for a clean background.
- Secondary color: Dark grey for text and important elements.
- Accent: Teal (#008080) to highlight interactive elements and the brand identity.
- Use a conversational UI layout with distinct sections for input and output.
- Ensure a responsive design for different screen sizes.
- Use simple and elegant icons for actions like copy, clear, and settings.
- Subtle animations to indicate loading or processing of code.

## Original User Request:
an app that takes code as input and provides with its explanation as the output. the ui needs to look modern,elegant and conversational
